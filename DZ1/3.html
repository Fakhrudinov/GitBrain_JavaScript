JavaScript lesson 1, task 3

*Чему будет равно JS-выражение 1000 + "108"?

Ответ

При сложении, если хотя бы один операнд является строкой, то второй будет также преобразован в строку.

Поэтому результатом будет конкатенация строк

При других арифметических операциях: '-*/', строковый оператор будет преобразован в число.
При удачном преобразовании получим математическую операцию.
При НЕудачном преобразовании результатом будет NaN
