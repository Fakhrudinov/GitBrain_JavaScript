Тут будут использоваться 2 типа алгоритмов сравнения - сравнение отношений и сравнение равенств

По алгоритму сравнения равенств (0 == null) в случае с null и числом - false возвращается как значение по умолчанию.

По алгоритму сравнения отношений (0 < null или 0 > null) null приводится к типу Number, c значением "+0" и получаем что 0 не больше(меньше) чем +0

По алгоритму сравнения отношений (0 <= null или 0 >= null) null приводится к типу Number, c значением "+0" и получаем что 0 не больше(меньше) чем +0, но второе условие на равенство тут выполняется, т.е. 0 == +0
