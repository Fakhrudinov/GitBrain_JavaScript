JavaScript lesson 2, task 1

1. Дан код:

var a = 1,
      b = 1,
      c,
      d;

c = ++a; alert(c); // 2
d = b++; alert(d); // 1
c = (2+ ++a); alert(c); // 5
d = (2+ b++); alert(d); // 4
alert(a); // 3
alert(b); // 3

Почему код даёт именно такие результаты?

Ответ

c = ++a; alert(c); // 2
Префиксная форма сначала делает вычисления, потом отдает значение переменной.
Т.е.: с = а + 1;

d = b++; alert(d); // 1
Постфиксная форма сначала отдает переменную, потом проводит с ней действия.
Т.е.: d = b; b = b + 1;

c = (2+ ++a); alert(c); // 5
Префиксная форма сначала делает вычисления, потом отдает значение переменной.
Т.е.: с = (2 + (a + 1));// где на начало вычислений 'а' уже было равно 2 из предыдущей строки.

d = (2+ b++); alert(d); // 4
Постфиксная форма сначала отдает переменную, потом проводит с ней действия.
Т.е.: d = (2 + b); b = b + 1;// где на начало вычислений 'b' уже было равно 2 из предыдущей строки.

alert(a); // 3
Для переменной 'а' было 2 инкремента в предыдущих строках.

alert(b); // 3
Для переменной 'b' было 2 инкремента в предыдущих строках.
